# MVVM Clean Architecture with SwiftUI

This project demonstrates a scalable implementation of **Clean Architecture combined with MVVM** in a SwiftUI application.

## Architecture Overview

The app is divided into four main layers:

- **Presentation**: SwiftUI views and ViewModels
- **Domain**: Business logic, entities, and use cases
- **Data**: Networking, DTOs, and repository implementations
- **Core**: Dependency Injection, routing, and utilities

Dependencies always point inward, ensuring loose coupling and testability.
